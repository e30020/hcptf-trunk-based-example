# hcptf-trunk-based-example

Example repository layout for a trunk-based Git strategy with HCP Terraform.

## Layout

```sh
.
├── dev
│   ├── main.tf
│   ├── outputs.tf
│   └── variables.tf
├── prod
│   ├── main.tf
│   ├── outputs.tf
│   └── variables.tf
└── stage
    ├── main.tf
    ├── outputs.tf
    └── variables.tf
```
